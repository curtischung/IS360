Reference Manual: <a> Tag

Abstract

The anchor tag creates a hyperlink which can be used to link information from in and outside of the website. The hyperlinks created can link to pictures, other pages on the websites, or other files, such as pictures, that exist on the website. They have plenty of other attributes with the <a> tag such as bookmarking, media, pings, etc, but the main usage is for hyperlinks.

Details

href

  • Specifies a hyperlink's destination
  • If href was not present in anchor tag, it will not become a hyperlink
  • Can point to:
    • An absolute URL
    • A relative URL
    • A specified ID in the website
    • A script
    • Other Protocols(https://, file, etc.)

download

  • Specifies a hyperlink file
  • When a user clicks on hyperlink, filename will be downloaded
  • NO restrictions on file extensions

target

  • Tells how the hyperlink will open in the browser
  • _blank opens in a new window/tab
  • _self opens in the same frame
  • _parent opens link in a parent frame
  • _top opens link in a full body window
  • framename opens the link in a frame specified by the writer

rel

rel specifies the relation between the current page/document to whatever is linked.

  • alternate gives an alternate version of the page
    • For example: print pages, translated, or mirrored
  • author gives the author of the document
  • bookmark us straightforward, and used to bookmark URLs
  • external links to a document that is not part of the same site
  • help gives the user a help document
  • license gives links to the licensing information for the document
  • next gives a link to the next document in the listing of the documents
  • prev gives a link to the previous documentin the listing
  • search links to a search tool in the page
  • tag gives a keyword for the document
